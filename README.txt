Intelligent Logistics Marketing Site (static)

How to run locally
1) Open a terminal in this folder
2) Run a static server, e.g.
   - Python:   python -m http.server 8080
   - Node:     npx http-server -p 8080

Then open:
http://localhost:8080/index.html

Assets
- assets/il-logo.png: provided logo
- assets/homepage-layout-reference.png: your reference screenshot (not used on the site)
- Dashboard visuals are SVG placeholders inside the HTML. Replace with real screenshots as needed.
